# &#x200B;5. Libreria di modelli

Un indice di riferimento rapido dei modelli di grafici di Firefly, organizzati in base a ciò che ciascuno di essi produce o fa. Ogni esempio è un punto di partenza: scambia il tuo marchio, il tuo prodotto e le tue richieste prima di utilizzare un modello in produzione.

## Generazione e stile delle immagini

| Modello grafico | Descrizione | Esempi di utilizzo |
|---|---|---|
| [**Guida introduttiva - Generare un&#39;immagine**](/help/firefly/graph/templates/get-started-gen-image.md) | Grafico di base con un nodo prompt in un nodo di generazione in un unico output. | <ul><li>Hero image</li><li>Test segnaposto</li><li>Immagine di esempio</li></ul> |
| [**Generazione di caratteri coerente**](/help/firefly/graph/templates/character-gen.md) | Carica un’immagine di riferimento di un personaggio, quindi scambia la scena o il prompt della posa per ogni nuova ripresa. | <ul><li>Mascotte ricorrente</li><li>Carattere raggi</li><li>Carattere istruttore</li></ul> |
| [**Estrazione stile**](/help/firefly/graph/templates/style-extraction.md) | Inserite un’immagine di riferimento per estrarne colore, luce e texture. | <ul><li>Trasferimento look</li><li>Abbinamento look stagionale</li><li>Abbinamento di umore</li></ul> |
| [**Atmosfera al tramonto**](/help/firefly/graph/templates/sunset-vibes.md) | Crea un’immagine tipografica 3D da un prompt di testo. | <ul><li>Sovrapposizione tagline</li><li>Slogan stagionale</li><li>Flash cespite vendita</li></ul> |
